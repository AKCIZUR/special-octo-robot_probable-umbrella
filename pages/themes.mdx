# Theme Packs

Tady je přehled 6 packů, které jsou součástí šablony.

## Packy

### Default
Čistý a univerzální layout s adaptivním dark modem.

### Glass
Translucentní cards, jemné blur efekty a moderní SaaS feel.

### GitHub
Content-first vzhled inspirovaný GitHub Docs.

### Nord
Chladná paleta a klidný technický styl.

### Dracula
Bohatý kontrast, fialová a růžová akcentace.

### Apple
Zaoblený system-like styl s elegantními detaily.
