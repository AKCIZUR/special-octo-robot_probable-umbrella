# Once UI Docs

Nextra template oblečený do čistého Once UI core kitu.

## Co je uvnitř

- GitHub Pages ready deployment
- automatický deploy po pushi do `main`
- jeden globální `custom-style.css`
- light i dark varianta pro každý pack
- 6 vizuálně odlišných UI kitů
- samostatný přepínač packu a režimu
