# Theme Studio

Moderní dokumentační template s jedním společným stylem a 6 theme packy.

## Co je uvnitř

- GitHub Pages ready deployment
- automatický deploy po pushi do `main`
- globální `custom-style.css`
- light i dark varianty pro každý pack
- 6 vizuálně odlišných UI kitů

## Použití

Vyber pack pomocí přepínače vpravo dole. Každý pack si pamatuje poslední volbu v `localStorage`.

## Co se dá upravovat

- barvy
- radius
- stíny
- navbar
- sidebar
- code block styl
- callout styl
- button styl
- celkové rozestupy
