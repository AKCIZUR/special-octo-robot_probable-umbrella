# Once UI Docs

Nextra template oblečený do čistého Once UI core kitu.

## Co je uvnitř

- GitHub Pages ready deployment
- automatický deploy po pushi do `main`
- lokální vývoj přes `pnpm dev`
- jeden globální `custom-style.css`
- light i dark varianta pro každý pack
- 6 vizuálně odlišných UI kitů
- přepínač packu a režimu s ukládáním do `localStorage`

## Jak to používat

Vyber theme pack a režim v plovoucím panelu vpravo dole. Všechny volby se uloží a po reloadu zůstanou zachované.
